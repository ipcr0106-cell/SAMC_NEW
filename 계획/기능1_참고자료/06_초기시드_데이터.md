## 6. 별표1~3 확장 전략

newsamc는 110건만 시드. 실제 별표1은 수천 건 규모.

- **Day 2 (개발계획서)**: 병찬이 `ingredient_list` 200개 입력
- 전략:
  1. newsamc의 110건을 SAMC_NEW 스키마로 변환 후 즉시 삽입
  2. `DB_최신/4_고시(공전)/식품공전_별표1.xlsx` 또는 HWP 원본에서 추가 추출
  3. 추출은 `law_extractor.py`(병찬)가 LLM 추출 → `is_verified=false` 저장
  4. 담당자 검수 후 `is_verified=true` 승격 (관리 UI 사용)
