  4. 프론트엔드 파일 수정 및 정리

     ┌─ 어떤 파일이 바뀌었나요? ──────────────────────────────────────┐

     ① 앱 전체 위치 이동
        전: frontend/candidate1/ 폴더 안에 Next.js 앱이 들어있었음
        후: frontend/ 바로 아래로 꺼냄 (일반적인 프로젝트 구조로 정리)

        이동된 것들
          package.json, tsconfig.json, next.config.ts   (앱 설정 파일)
          app/              (모든 페이지)
          components/       (공통 레이아웃 컴포넌트)
          public/           (이미지·아이콘)

     ② 기능5 페이지 경로 이름 변경
        전: app/cases/[id]/feature5/page.tsx
        후: app/cases/[id]/label/page.tsx

        이유: 'feature5'는 개발용 이름이고, 실제 기능이
              '한글표시사항(label)'임을 명확히 하기 위해 변경

     ③ API 호출 파일 위치 이동
        전: frontend/services/apiClient.ts
        후: frontend/lib/api.ts

        이유: Next.js 관례상 공통 유틸 파일은 lib/ 폴더에 두는 것이 표준

     ④ 임시 폴더 삭제
        frontend/candidate1/ 폴더 (파일 다 옮기고 나서 제거)
